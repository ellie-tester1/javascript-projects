# JavaScript Basics Demo

This project demonstrates basic JavaScript concepts and DOM interaction.

## Features
- Working with arrays
- Filtering even numbers
- Using loops
- Creating simple objects
- Handling button click events

## Technologies
- HTML
- CSS
- JavaScript

## How it works
1. The script filters even numbers from an array.
2. Displays information about the author in the console.
3. Uses loops to print numbers from 1 to 10.
4. Handles a button click event.

When the button is clicked, a message appears in the browser console.
