User Data Collector

This project is a simple JavaScript application that collects user information (name, age, and email) through prompts and logs the results to the browser console.

Features
    •    Prompts the user to enter their name, age, and email.
    •    Stores each user’s data as an object in an array.
    •    Displays the collected data in the console.

Technologies
    •    HTML
    •    JavaScript

How to Run
    1.    Open index.html in your browser.
    2.    Follow the prompts to enter the user information.
    3.    Open the browser console to view the collected data.
