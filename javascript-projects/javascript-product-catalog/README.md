# JavaScript Product Catalog

This project displays a list of smartphones using JavaScript.  
Each product is shown as a card with an image, name, description, price, and quantity.

## Features
- Displaying product cards dynamically
- Showing product image, name, and description
- Displaying product price
- Displaying product quantity
- Responsive card layout

## Technologies
- HTML
- CSS
- JavaScript
