# Kyrgyzstan 7 Regions

This project displays information about the 7 regions of Kyrgyzstan using JavaScript.

## Features
- Region cards with images
- Region description and history
- Population information
- Dynamic rendering using JavaScript

## Technologies
- HTML
- CSS
- JavaScript

## Project Structure

kyrgyzstan-7-regions  
├ index.html  
├ style.css  
├ script.js  
├ data.js  
└ README.md  

## How to Run

1. Download or clone the project  
2. Open the project folder  
3. Open index.html in your browser
